# Customer Behavior Analysis

An end-to-end data analytics project that explores customer purchase behavior using Python, PostgreSQL, and Tableau. The project demonstrates data preprocessing, SQL-based analysis, and visualization to extract meaningful insights from raw data.

## Workflow
- Raw CSV data ingestion
- Data cleaning and EDA using Python (Jupyter Notebook)
- Data storage and querying using PostgreSQL
- Dashboard creation using Tableau

## Skills Demonstrated
- Data Cleaning & Transformation
- Exploratory Data Analysis (EDA)
- SQL Analytics
- Business Insight Generation
- Data Visualization

## Tools Used
- Python (Pandas)
- PostgreSQL
- SQL
- Tableau
- Jupyter Notebook

## Use Case
Understanding customer behavior patterns to support data-driven business decisions.
